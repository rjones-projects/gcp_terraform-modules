# project_iam YAML examples

These examples show how to configure the `project_iam` module via YAML and
convert it to `terraform.tfvars.json` with `yaml_to_tfvars.py`.

## Usage

- Update paths and values if needed.
- From the repo root, run:

```bash
python3 .github/scripts/yaml_to_tfvars.py modules/project_iam/examples/basic.yaml deploy/terraform.tfvars.json
```
