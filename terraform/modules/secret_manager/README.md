# Google Cloud Secret Manager

This module creates multiple Google Secret Manager secrets in one run using
`secret_manager.spec` from `terraform.tfvars.json` generated by `yaml_to_tfvars.py`.

## Features

- Create many secrets from one `spec` list
- Automatic or user-managed replication per secret
- Optional secret version creation:
  - use `secret_data`
  - or generate a random password (`generate_random_password`)
- Optional IAM:
  - default accessor binding via `create_binding` + `members`
  - arbitrary role/member pairs via `iam_bindings`

## Usage with `project.yaml`

```yaml
secret_manager:
  source: "../modules/secret_manager"
  depends_on: []
  spec:
    - secret_id: "db-password"
      replication_automatic: true
      generate_random_password: true
      labels:
        owner: "platform"
    - secret_id: "api-token"
      replication_automatic: false
      replicas:
        - location: "europe-west2"
        - location: "europe-central2"
      secret_data: "replace-me"
      iam_bindings:
        "roles/secretmanager.secretAccessor":
          - "group:team@example.com"
```

`yaml_to_tfvars.py` passes `secret_manager = var.secret_manager` in `deploy/main.tf`.
This module reads `var.secret_manager.spec` in `locals.tf`.

## Minimal Terraform usage

```hcl
module "secret_manager" {
  source     = "../modules/secret_manager"
  project_id = var.project_id
  region     = var.region
  secret_manager = {
    spec = [
      {
        secret_id                = "example-secret"
        replication_automatic    = true
        generate_random_password = true
      }
    ]
  }
}
```

## Input

Top-level input:

- `project_id` (`string`, required)
- `region` (`string`, optional compatibility input)
- `secret_manager` (`any`, expected shape below)

Expected `secret_manager.spec` item attributes:

- `secret_id` (string, required)
- `labels` (map(string), optional)
- `expire_time` (string RFC3339, optional)
- `ttl` (string duration, optional)
- `replication_automatic` (bool, default `true`)
- `kms_key_name` (string, optional; used for automatic replication CMEK)
- `replicas` (list, optional; used for user-managed replication)
- `rotation` (object, optional)
- `topics` (list(string), optional)
- `generate_random_password` (bool, default `false`)
- `random_password_length` (number, default `32`)
- `random_password_special` (bool, default `true`)
- `random_password_override_special` (string, default `_%@`)
- `secret_data` (string, optional)
- `create_binding` (bool, default `false`)
- `members` (list(string), optional; used with `create_binding`)
- `iam_bindings` (map(list(string)), optional)

## Outputs

- `secrets`: map of created secret resources
- `secret_resource_ids`: map `secret_id => resource id`
- `secret_version_resource_ids` (sensitive): map `secret_id => version id`
